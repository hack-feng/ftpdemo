# ftpdemo
上传ftp服务器util工具类
```
  com.maple.ftpdemo
    -bean
      -FtpBean.java             # ftp基础配置对象
    -util
      -FtpUtil.java             # 上传ftp服务器工具类
      -GenerateDirUtil.java     # 自动生成Ftp服务器存放文件的目录 
    -FtpDemo.java               # Ftp上传测试用例
```
